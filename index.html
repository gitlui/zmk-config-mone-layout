MONE LAYOUT by gitlui

This is my daily driving layout that I have been using since early 2023. It is designed to be used with a 40% keyboard (ferris sweep) on a MAC. If you want a windows/linux version just request it as I had that before. There have been some iterations first, but now it has been quite stable.

I use the colemak layout but the alpha layer can also be used with qwertz or qwerty. I tried to use it with qwerz but didn't know what key to put where the 'o' in colemak is :D

Attention: The keymap in the code assumes that you have a german keyboard layout set in your system. If you want to use it like it is shown here tell me and I'll upload the corresponding layout too.
